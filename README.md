## 📱 Tentang Aplikasi

SIAP JALAN adalah aplikasi Progressive Web App (PWA) untuk backup dan monitoring aduan serta penanganan jalan. Aplikasi ini terintegrasi dengan Google Sheets dan dapat diinstal di perangkat HP maupun PC.

### ✨ Fitur

- 🔐 **Login Sistem** - Multi-role (Admin, Operator, Pegawai)
- 📝 **Input Aduan** - Tambah laporan baru dengan tim dan PIC
- 📋 **Daftar Aduan** - Lihat, filter, dan edit aduan
- 📊 **Dashboard** - Statistik, peringkat tim, dan monitoring
- 📑 **Rekap Pekerjaan** - Data proyek dan kontrak
- 👥 **Master User** - Kelola pengguna sistem
- 📱 **PWA Support** - Install di HP/PC
- 🔄 **Auto Sync** - Backup otomatis setiap 30 menit

## 🚀 Cara Menggunakan

### Instalasi PWA

#### 📱 Android (Chrome)
1. Buka aplikasi di Chrome
2. Tap ikon menu (⋮) di pojok kanan atas
3. Pilih "Install app" atau "Add to Home screen"
4. Klik "Install"

#### 📱 iOS (Safari)
1. Buka aplikasi di Safari
2. Tap ikon Share (⬆️)
3. Scroll ke bawah dan pilih "Add to Home Screen"
4. Klik "Add"

#### 💻 PC (Chrome/Edge)
1. Buka aplikasi di browser
2. Klik ikon install di address bar
3. Klik "Install"

### Role & Akses

| Role | Akses |
|------|-------|
