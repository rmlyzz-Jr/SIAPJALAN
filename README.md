# 🚀 SIAP JALAN - Sistem Integrasi Aduan dan Penanganan Jalan

## Deskripsi
SIAP JALAN adalah aplikasi berbasis web untuk mengelola aduan dan penanganan jalan dengan fitur **Progressive Web App (PWA)** yang memungkinkan diakses secara offline dan diinstall sebagai aplikasi di perangkat mobile.

## Fitur Utama
- ✅ **Login Multi-role**: Admin, Operator, Pegawai
- ✅ **Input Aduan**: Menambahkan aduan baru dengan disposisi otomatis ke BINAMARGA
- ✅ **Daftar Aduan**: Filter, pencarian, edit, dan hapus aduan
- ✅ **Dashboard**: Statistik, peringkat tim, grafik bulanan
- ✅ **Master User**: Kelola pengguna (Admin only)
- ✅ **PWA Support**: Install sebagai aplikasi, offline cache, push notification

## Teknologi
- HTML5, CSS3, JavaScript
- Google Apps Script (Backend)
- Google Sheets (Database)
- Progressive Web App (PWA)
